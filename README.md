# 🚀 Getting Started

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Drash011/Tic-Tac-Toe
```

## 2️⃣ Open Project Folder

```bash
cd Tic-Tac-Toe
```

## 3️⃣ Run Project

Open `index.html` in your browser.

# 🎮 How To Play

- Choose game mode
- ❌ and ⭕ take turns
- Match 3 symbols to win
- Horizontal, Vertical, or Diagonal
